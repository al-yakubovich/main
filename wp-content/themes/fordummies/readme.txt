== Description ==

For Dummies is a non intimidating and easy-to-use WordPress theme with responsive design (optimal viewing and interaction with a wide range of devices – PC/Tablet/SmartPhone). Online guide, automatic suggested plugins installer and only 1 click demo install.  ForDummies is a modernized take on an ever-popular WordPress layout — the horizontal masthead with an optional right sidebar that works perfectly for blogs and websites. It has custom color options (also menu) with beautiful default color schemes (black or white skin with only 1 click), 3 footer widgets, layout Full or boxed. WooCommerce compatible. Dedicated Search Results and 404 page. Social menu available. Header phone, email and hours. Background opacity control, Footer Copyright configurator, back to top button and a lot more. Included slider, Anti Hacker protection, page loading image and a lot of shortcodes. Multilingual ready, English and French files included.  Anti Hacker Protection plugin included. Visit our demo site (www.ThemeForDummies.com) for updated Online Guide e more.



== Installation ==



1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.

2. Type in For dummies in the search form and press the 'Enter' key on your keyboard.

3. Click on the 'Activate' button to use your new theme right away.

4. Go to https://codex.wordpress.org/For_dummies for a guide on how to customize this theme.

5. Navigate to Appearance > Customize in your admin panel and customize to taste.
