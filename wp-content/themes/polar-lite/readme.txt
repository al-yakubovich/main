Polar Wordpress Theme
---------------------
Polar is a minimal and responsive Sueva child theme, based on Bootstrap framework, with a different layout for elements and sidebar.

Created by ThemeinProgress, https://www.themeinprogress.com 

Important: This version of Polar Lite (3.0.0) needs the same version of the parent theme Suevafree (3.0.0)
