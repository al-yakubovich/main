Getting Started
===============

How to Install the Theme?
-------------------------
1. Go to WP Admin > Appearance > Themes > Add New > Upload.
2. Upload the puremag.zip file, in which you found this readme.txt
3. Click on Install and Activate.

Setting it Up
-------------

Configure theme settings using WordPress customizer.

Dashboard > Appearance > Customizer
